# 归宫表(Karya Bhava)——问什么事,看哪个宫

> 事主星(karya lord)= 该宫的宫主。判成败的核心 = 命主星/月亮 与 事主星的连接。

## 主表

| 问题类型 | 主宫(karya) | 辅看 |

## 三个固定角色(每张盘都先标出)

- **命主星(L1)= 问主本人**。它的状态=你在这件事里的位置与姿态。
- **月亮 = 问主的心 + 事态的流向**。prashna 里月亮权重极高,近似"第二命主"。
- **7宫 = 交易/对局的"对方"**(房东、卖家、对手、合作方、谈判对象)。注意:同一张盘里 7宫可能同时是"伴侣"和"交易对方"——两层都读,分开写。

## 选宫纪律

- 先定主宫,再看辅宫;结论以主宫为准,辅宫只补细节。
- 问题跨两件事(如"换工作并搬城市") → 让用户拆成两问,分别归宫(可同一张盘里分别读,但判词分开)。
- 拿不准归宫时,回到问题的"落点":用户最终要的结果发生在哪个生活领域,就归哪个宫。
