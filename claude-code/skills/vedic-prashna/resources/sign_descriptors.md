## 配套修饰(叠加在星座词条上)

- **karya 宫主旺/庙** = 里子好:"气质纵有将就,内核质地好"。宫主陷/敌 = 反之。
- **凶星相位入 karya 宫的具体化**(以房屋为例):
  - 土星照 → 旧损/负担/条款重;土星自水象星座照 → 特查**水**(管道/渗漏/浴室)。
  - 火星照 → 火与力系统(灶/热/电路),及**竞争**;火星在 Krittika 等火宿 → 灶/热源加倍留意。
  - Rahu 照 → 照片/宣传与实物有落差,虚象成分,眼见为实。
  - Ketu 照 → 有"缺一块"(缺配套/缺证件/某功能缺失)。
- **金星状态** = 卖相/舒适度;**木星状态** = 空间/格局/"住进去的滋养感"。
- 词表与实物对照后,**命中与否都写进台账**——词表的长期命中率也是要被审计的对象。
